## 已知問題

1. **滑桿 VIZ 文字裁切** — 第 4 頁滑桿在極端比例（<10% 或 >90%）時，文字可能與邊界重疊。可在 `.viz-container` 調整 `height` 或縮小字體。

2. **Pollinations.ai 生圖品質不穩定** — 背景底圖使用免費 API，每次生成結果不同。若需要穩定品質，可改用自備 API（OpenRouter / OpenAI）並修改 `scripts/draw.py`。

3. **Reveal.js 5.1 CDN** — 依賴 jsDelivr CDN，離線環境無法使用。可下載後改為本地引用。
